This website will assist users in booking movie tickets across India in their favorite theaters. Unlike many other movie ticket websites that simply display movie titles, our platform provides valuable information such as IMDb ratings, trailers, and user reviews. Additionally, users can compare ticket prices from different theaters nearby to find the most suitable option for themselves.
System Architecture
Frontend → Backend (API) → Database
Tech stack:
Frontend: React.js with React Router for page navigation
Backend: Node.js + Express
Database: MongoDB (non-relational) / MySQL (relational)
Authentication: JWT-based login/signup
Hosting:
Frontend → Vercel
Backend → Vercel
Database → MongoDB Atlas /MySQLWorkBench
